Color Schema

  • Primary Color: Charcoal Gray (#2E2E2E) For backgrounds and sections to provide a strong yet neutral foundation.
  • Accent Color: Crimson Red (#D72638) Used sparingly for call-to-action buttons, highlights, or important elements.
  • Secondary Color: Jet Black (#141414) For main text and deep contrast elements.
  • Secondary Background: Pure White (#FFFFFF) To balance the darkness and provide a clean, readable layout.
  • Soft Gray: Ash Gray (#B0B0B0) For subtle elements, borders, and secondary text.

Typography

  • Heading: Inter
  • Body Text: Roboto
  • Code Blocks: Fira Code
